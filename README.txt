Usuário
=======
- Username
- Senha (obviamente criptografada)
- Nome completo
- Nº USP
- Grupo (curso)
- Data de cadastro
- Quota de disco
- Quota de impressão
- Administrador?

Máquina
=======
- Hostname
- Endereço IP
- MAC Address
- Sala (local)
- Tipo (cliente/servidor)

Impressora
==========
- Nome
- Endereço IP
- Marca
- Modelo
- Sala (local)

Pacote
======
- Nome
- Versão instalada
- Dependências (outros pacotes)
